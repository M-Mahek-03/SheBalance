# How to Add Screenshots to README

## Required Files

You need to add these 4 screenshot files to the root directory:

1. `Dashboard.jpeg` - Screenshot of the artisan dashboard
2. `AI Sakhi.jpeg` - Screenshot of AI Sakhi chatbot
3. `SkillScan.jpeg` - Screenshot of SkillScan feature
4. `Digital Twin.jpeg` - Screenshot of Digital Twin feature

## Steps to Add Screenshots

1. Save your screenshot files with the exact names above (including spaces)
2. Place them in the root directory: `SHE-BALANCE-main/SHE-Balnce-main/`
3. Run these commands:

```bash
cd SHE-BALANCE-main/SHE-Balnce-main
git add "Dashboard.jpeg" "AI Sakhi.jpeg" "SkillScan.jpeg" "Digital Twin.jpeg"
git commit -m "Add platform screenshots"
git push
```
